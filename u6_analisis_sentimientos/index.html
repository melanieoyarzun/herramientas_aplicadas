Paso 2
Tokenizar el texto

Tokenizar significa separar el texto en unidades más pequeñas, como palabras. Esa operación es una base importante del procesamiento de lenguaje.

Tokenización

reviews_tokens <- reviews %>%

Paso 3
Asignar un puntaje de sentimiento base

En este punto aparece una lógica clásica: cruzar palabras con un diccionario que clasifica términos como positivos o negativos.

Sentimiento léxico 
# Unimos palabras con el diccionario de sentimientos
reviews_sentiment_bing <- reviews_tokens %>%
  inner_join(sentiments, by = "word")

# Sumamos puntajes por reseña
reviews_scored <- reviews_sentiment_bing %>%
  mutate(value = ifelse(sentiment == "positive", 1, -1)) %>%
  group_by(customer) %>%
  summarize(sentiment_score = sum(value))

head(reviews_scored)
# Score the reviews with a small positive/negative lexicon
lexicon = {
    "excellent": 1, "recommend": 1, "good": 1, "better": 1,
    "perfect": 1, "slow": -1
}

scores = (
    tokens.assign(value=tokens["word"].map(lexicon).fillna(0))
    .groupby("customer", as_index=False)["value"]
    .sum()
    .rename(columns={"value": "sentiment_score"})
)

Paso 4
Corregir negaciones y volver a puntuar

Una forma simple de mejorar la lectura es detectar bigramas de negación y reemplazarlos por expresiones más coherentes con el sentimiento real.

Negaciones 
# Buscamos negaciones frecuentes
negaciones <- reviews %>%
  unnest_tokens(bigram, review, token = "ngrams", n = 2) %>%
  count(bigram, sort = TRUE) %>%
  separate(bigram, c("word1", "word2"), sep = " ") %>%
  filter(word1 %in% negation.words & !word2 %in% stop_words$word) %>%
  inner_join(get_sentiments("bing"), by = c(word2 = "word"))

# Reemplazamos y repetimos el scoring
review_negations_fix_manual <- mgsub(reviews$review, c("expected better quality", "did not meet my expectations", "not satisfied"), c("bad", "bad", "bad"))
# Apply a small manual negation fix before rescoring
replacements = {
    "not satisfied": "bad",
    "not recommend": "bad",
    "did not meet my expectations": "bad",
    "expected better quality": "bad"
}

reviews_fixed = reviews.copy()
for old, new in replacements.items():
    reviews_fixed["review"] = reviews_fixed["review"].str.replace(old, new, regex=False)

Paso 5
Mirar palabras frecuentes

Además del puntaje, también sirve mirar qué palabras aparecen más veces después de sacar stopwords y términos poco informativos.

Frecuencias 
# Contamos palabras relevantes
freq.table <- reviews %>%
  unnest_tokens(word, review) %>%
  anti_join(stop_words) %>%
  anti_join(ignore.words) %>%
  count(word, sort = TRUE)

head(freq.table)
# Count frequent words after a simple stopword filter
from collections import Counter

stop_words = {"the", "i", "it", "to", "with", "am", "was", "but", "this"}
