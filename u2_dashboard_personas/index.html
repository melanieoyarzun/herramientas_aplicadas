Aplicación Unidad 2

Dashboard interactivo de Gestión de Personas

Este artefacto usa datos sintéticos para mostrar cómo un KPI, una serie temporal, una comparación entre áreas y una distribución pueden cambiar la lectura de un mismo fenómeno.

El foco está puesto en contrataciones, rotación y capacitaciones, incluyendo participación en cursos de inglés y Excel.

¿Qué mirar aquí?

Aquí puedes explorar un mini dashboard y observar cómo cambia la lectura cuando no miramos solo un número, sino también su evolución, su distribución y sus diferencias entre áreas.

  • Elige una métrica principal.
  • Prueba distintos cálculos, como promedio o mediana.
  • Compara áreas y abre la distribución detrás del KPI.
  • Explora relaciones posibles entre variables, con cautela.

Pregunta para explorar

Si solo miraras la rotación promedio, ¿qué podrías estar dejando fuera? Prueba cambiar la métrica, abrir la distribución y comparar áreas antes de responder.

Comparación entre áreas

Permite ver diferencias entre áreas para un mes específico y cambiar la forma de comparación: barras para una lectura rápida, boxplots para resumir la variación o distribuciones para ver cómo se reparten los valores en cada área.

Distribución detrás del KPI

Ordena todos los valores observados en las áreas seleccionadas y muestra media y mediana. La idea es abrir lo que el resumen no alcanza a mostrar.

Cruce exploratorio entre variables

Este gráfico sugiere patrones entre dos métricas, por ejemplo entre rotación y asistencia a capacitación. Sirve para abrir preguntas, no para afirmar causalidad.
